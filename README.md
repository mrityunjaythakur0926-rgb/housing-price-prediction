# 🏠 Housing Price Prediction System
A Machine Learning model designed to predict residential property prices based on key physical and locational attributes.

## 🚀 Project Overview
This project demonstrates a complete ML pipeline—from data generation and exploratory analysis to model training and evaluation. [cite_start]It uses **Linear Regression** to estimate house prices.

## 📊 Key Features
* [cite_start]**Multi-variable Analysis:** Predicts prices based on Area (sqft), Bedrooms, Age of the house, and Location Score[cite: 1].
* [cite_start]**Data Visualization:** Includes a correlation heatmap to show how features like 'Area' impact 'Price'[cite: 2].
* [cite_start]**Model Performance:** Evaluated using Mean Absolute Error (MAE), Root Mean Squared Error (RMSE), and R2 Score[cite: 3].
* [cite_start]**Interactive Interface:** Features a Python function for real-time price estimation for new data points[cite: 3].

## 🛠️ Tech Stack
* **Language:** Python
* [cite_start]**Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn [cite: 1]

## 📂 How to Run
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Run the script: `python housing_model.py`.
